# 1. 옷 특징과 관련된 가중치 기준 정하기 -> 설문조사 제작
- 직장인 대상 옷 기억력 설문조사
=> 생각보다 많은 사람들이 가까운 사람의 옷을 기억함
- 4반 대상 컨설턴트님 옷 기준으로 설문 실시
=> 아직 참여율이 저조해 어느정도 결과 취합 후 통계 예정

⇒ 설문조사 결과 활용 (Q**5. 기억이 난다면, 어떤 이유로 생각하시나요?)**
- 색상 (채도)
- 색의 다채로움 정도
- 패턴, 프린팅
- 스타일 : ex) 반팔+셔츠+바지….
