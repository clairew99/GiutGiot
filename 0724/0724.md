## 화면 별 기능
1. 메인 페이지 
- 기억에 남아있는 옷

- 기억에 남아있지 않은 옷

- 옷 수정할 수 있도록

- 음성 활성화

2. 캘린더
- 월간 / 2주치

- 내가 무슨 옷을 입었는지에 대한 데이터

- 데이터에 대한 분석값 (캘린더 클릭 ?) 

- 간단한 상세 정보 

3. 옷 상세 정보 
- 그날 입은 옷에 대한 정보 (색상, 패턴, 종류)

- 분석값을 여기에다 넣어도 될 듯

4. 상세 정보 CRUD
- 옷 상세 정보 화면과 비슷하게 구성

- 기존에 입은 옷 수정

- 새로운 옷 생성 - 팔레트, 키워드 등 사용자 선택 받기

5. 음성 활성화 
 

6. 설정
- 회원 정보 수정

- 회원 탈퇴

- 튜토리얼

  - 우리의 음성 데이터는 텍스트화 안된다는 걸 말해주기!
